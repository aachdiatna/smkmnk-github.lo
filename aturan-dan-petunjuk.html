Formulasi CMS - Free Opensource CMS for education website

Silahkan Anda menggunakan software ini untuk membangun sebuah situs sekolah, Anda dapat mendesain ulang script maupun tampilan pada CMS ini sesuai dengan kebutuhan anda, memperbayak jumlah copy atau mendistribusikan CMS ini. Dengan catatan tidak menghapus link developer dan link ke situs resmi Formulasi CMS, dan kami rasa itu bukan masalah untuk anda karena link yang ada tidak membutuhkan space yang banyak.
Tidak ada garansi dari kami jika anda mengalami error atau merasa rugi ketika menggunakan Formulasi CMS versi free, anda hanya dapat memberikan feedback yang berisi laporan error. Garansi hanya dapat diperoleh dalam versi premium**, dengan syarat dan ketentuan yang berlaku.

**Versi premium adalah versi CMS dimana anda memesan CMS ini secara unik, dimana ada penambahan modul atau fungsi sistem dan penambahan tema unik sesuai dengan keinginan anda. Untuk mendapatkan versi premium anda dapat menghubungi suport dari Formulasi CMS atau klik disini.

Minimal sistem yang dibutuhkan dalam penggunaan Formulasi CMS

Dalam pembuatan dan testing sistem ini kami menggunakan beberapa Sistem operasi dan Web browser, karena keterbatasan sarana yang ada, untuk sementara CMS ini berjalan normal pada spesifikasi Sistem seperti berikut :

  • Semua sistem operasi PC atau Notebook yang mendukung web server.
  • Web Browser seperti : Mozilla Firefox minimal versi 3.6 (Rekomendasi), Google Chrome, Safari, Untuk ms Internet Explorer tidak bekerja maksimal, karena beberapa javascript dan CSS 3 tidak berjalan dalam browser ini.
  • XAMPP minimal minimal versi 1.7.3 (jika dijalankan pada lokal server)
  • CMIIW :p
